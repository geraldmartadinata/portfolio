## 📁 Projects Overview

### 1. Interactive Personal Portfolio
A front-end web application built to showcase my profile, technical skills, and projects. The interface is designed to be highly interactive, utilizing scroll-driven animations and dynamic theme adjustments to provide an engaging user experience.

**Key Features:**
- **Scroll-Driven Animations:** Implemented complex, reversible scroll animations for layout transitions and 3D card flips using GSAP.
- **Dynamic Theming:** Automated light and dark mode transitions based on the user's viewport and scroll progress.
- **Interactive Audio Player:** Custom background music player complete with reactive, synchronized soundwave animations.
- **Responsive Design:** Optimized layout for seamless viewing across desktop and mobile devices.

**Tech Stack:**
- HTML5 & CSS3
- Vanilla JavaScript
- Tailwind CSS
- GSAP (GreenSock Animation Platform) & ScrollTrigger

### 2. Stock Updater with MongoDB
A back-end inventory management service built to handle real-time stock updates and data synchronization. This project demonstrates foundational skills in database management and server-side logic.

**Key Features:**
- **Database Integration:** Direct connection to a MongoDB database to securely store and retrieve inventory records.
- **CRUD Functionality:** Endpoints designed to Create, Read, Update, and Delete stock items efficiently.
- **Data Synchronization:** Ensures inventory numbers are consistently accurate and updated in real-time across transactions.

**Tech Stack:**
- Node.js
- Express.js
- MongoDB & Mongoose
